# QA Checklist — CMS

## Posts
- [ ] Créer un post avec un featured media (sélection depuis la Media Library).
- [ ] Insérer un média inline via le Media Picker dans l’éditeur.
- [ ] Sauvegarder un brouillon puis publier le post.
- [ ] Utiliser le changement rapide de statut depuis la liste.

## Media
- [ ] Uploader un média via la Media Library.
- [ ] Vérifier que le média est réutilisable dans l’éditeur.

## Overview
- [ ] Vérifier que les KPI correspondent aux listes (posts, drafts, published, scheduled, events, media, users, comments).
- [ ] Cliquer sur chaque KPI et vérifier la navigation vers la page filtrée.

## Permissions
- [ ] Vérifier que les erreurs 403 affichent un message “Accès admin requis”.
- [ ] Vérifier qu’un 401 redirige vers la page de login.
